## Need Help?

If you encounter issues:

1. **No remote?** Add it:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/netsec-toolkit.git
   ```

2. **Authentication?** Use SSH or token (see QUICK_PUSH.md)

3. **Branch name?** Check with:
   ```bash
   git branch
   ```
